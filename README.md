# 🌐 web-page-at1
Projeto web de autenticação utilizando Node.js da disciplina de Desenvolvimento Web, com o objetivo de aprender a utilizar a biblioteca Express para criação de servidores e definição de rotas.

---

## 📌 Sobre o projeto

Este projeto consiste na construção de uma página web estática, explorando a estruturação de conteúdo com HTML e estilização com CSS.

A proposta tem como foco o aprendizado prático de:

- Estrutura básica de páginas HTML
- Organização de conteúdo
- Estilização com CSS
- Publicação utilizando GitHub
